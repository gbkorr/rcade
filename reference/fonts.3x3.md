# 3x3 Font

Basic
[font](https://gbkorr.github.io/rcade/reference/render.makefont.md) with
no lowercase. Readability is poor, but this produces the most compact
text possible.

Use this with
[`render.text()`](https://gbkorr.github.io/rcade/reference/render.text.md).

## Usage

``` r
fonts.3x3
```

## Format

A [font](https://gbkorr.github.io/rcade/reference/render.makefont.md); a
list containing font-related properties and a list of sprites for each
character.
